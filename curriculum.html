CodeCraft Academy Python Class

At CodeCraftAcademy we believe that children should get involved in computer science as early as possible. Through hands-on experience, and an adaptive curriculum, we will take your child from an absolute beginner to be able to do projects such as these



Hangman
TicTac Toe
Choose Your Adventure
Snake Game
Rock, Paper, Scissors
Guess the Number
Rocketship Animation
Mad Libs
Topics you will learn in this class, from beginner to more advanced:
What is coding, and what is Python?
Input and output
Proper syntax
Data Types(int, bool, float, string, char)
Functions (and their uses)
Data Structures (Lists, Tuples, Dictionaries) and their uses
Loops (for and while)
Libraries(Common libraries in python, how to use them, and what libraries are used for)
Python Turtle, Tkinter
OOP(Object Oriented Programming)- Classes, objects, inheritance, polymorphism
